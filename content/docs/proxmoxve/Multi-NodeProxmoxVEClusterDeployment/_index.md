- Designed and deployed a multi-node Proxmox VE cluster using Corosync.
- Configured cluster quorum and node membership for high availability environments.
- Added and managed multiple Proxmox nodes within a centralized datacenter architecture.
- Prepared shared storage and network configurations for cluster-wide resource management.
- Configured ZFS-backed storage across nodes to support replication strategies.
- Implemented cluster networking, SDN integration, and centralized configuration management.
- Evaluated quorum requirements and fault tolerance strategies for virtualization clusters.
