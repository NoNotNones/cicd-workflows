### Network Virtualization and SDN
- Configured Linux bridge networking (vmbr) for virtual machines and containers.
- Designed and managed virtual networking infrastructure using Proxmox SDN.
- Created isolated virtual networks through SDN Zones and VNets.
- Implemented NAT-based private networks using SNAT and DHCP services.
- Configured subnetting, gateways, and IP address management for virtualized workloads.
- Designed network segmentation strategies for management, storage, migration, and cluster traffic.
- Implemented virtual networking solutions for lab and multi-node cluster environments.
- Evaluated network performance considerations including bandwidth utilization, replication traffic, and live migration workloads.
