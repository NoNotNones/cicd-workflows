# PURPOSE
## Details : 
-	Manage and operate Azure infrastructure (App Service, VMs, Azure SQL, APIM, Dynamics 365) 
-	Design secure architectures using Application Gateway (WAF), Azure Firewall, NSG, Private Endpoint 
-	Implement CI/CD pipelines using Azure DevOps or GitHub 
-	Automate infrastructure using Terraform / Bicep / ARM 
-	Monitor systems using Azure Monitor, Log Analytics 
-	Optimize cost, performance, and autoscaling 
-	Ensure High Availability (HA) and Disaster Recovery (DR) 
-	Implement Zero Trust and identity-based security 
-	Manage API integrations and service connectivity across Azure
